## 备注

### 名词解释

- **正常网络**: 指能正常访问 [Google](https://www.google.com/), [Wikipedia](https://www.wikipedia.org/) 等的网络.

- **校园网**: 指我校校园网, 可直接访问 [图书馆](https://lib.jlu.edu.cn/) 等校内网站. 与外面推销的 `校园手机卡` 无关.

- **Python**: 若无特殊说明, 均指 [Python3](https://www.python.org/) .
